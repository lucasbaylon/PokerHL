Sélection du mode d'entraînement

Choisissez le mode d'entraînement auquel vous souhaitez jouer.

Infini

Répondez aux situations qui se présentent à vous de manière infinie, et stoppez quand vous le souhaitez.

Turbo

Un mode basé sur la rapidité. Répondez à un maximum de situations dans le temps imparti.

Défi

Répondez à un nombre prédéfini de situations pour compléter l'entraînement. A la première erreur, c'est perdu.
